### 使用ケース
本規則で用いられるケースを以下にまとめる

|ケース名|説明 |例 |
|:---|:---|:---|
|スネークケース | 単語間をアンダースコア「_」で連結| str_user_name |
|アッパースネークケース |全て大文字で単語間をアンダースコア「_」で連結 |USER_NAME |
|キャメルケース |連結する際に単語の始めを大文字にして連結 |strUserName |
|パスカルケース |先頭の単語を含め、単語の始めを大文字にして連結 |StrUserName |

### 変数
- 使用ケース  
  スネークケース（*snake_case*）  

- 補足  
変数名には可読性を向上させるため、先頭に型名の略称を記述する（略称は下記参照）

    |型名 |略称 |例 |
    |:---|:---|:---|
    |Int(Int8, Int16等含む) |int |int_hoge |
    |UInt(Uint8, UInt16等含む) |unt |unt_hoge |
    |Float |flt |flt_hoge |
    |Double |dbl |dbl_hoge |
    |Bool |bln |bln_hoge |
    |Character |chr |chr_hoge |
    |String |str |str_hoge |
    |Array |arr |arr_hoge |
    |Dictonary |dic |dic_hoge |
    |Set |set | set_hoge |

- 例
    ```
    // Int  
    int_number: Int = 5  

    // Float  
    flt_number: Float = 1.4  

    // Double  
    dbl_number: Double = 1.2  

    // Array  
    arr_car: [String] = ["hoge", "huga"]  

    // Dictionary  
    dic_car: [String: String] = ["hoge": "huga"]  

    // Set  
    set_car: Set<String> = ["hoge", "huga"]  
    ```

  #### グローバル変数
  極力実装しないことが望まれるが、やむを得ない場合は先頭に「g_」を加える

  - 例  
      ```

      ```

### 定数
- 使用ケース  
  アッパースネークケース（*UPPER_SNAKE_CASE*）  

- 補足  
  基本的に定数名に型名は不要

- 例  
    ```
    // 良い例  
    PI = 3.14159265359  
    FILE_PATH = "/a/b/c/Documents/README.md"  

    // 悪い例(基本的に先頭の型名は不要)  
    DBL_PI = 3.14159265359  
    STR_FILE_PATH = "/a/b/c/Documents/README.md"  
    ```

### 対応必須
エラーやパフォーマンス低下の原因になり得る項目は対応必須とし、下記に示す

- モジュール名の省略禁止  
Importしたモジュールの変数、関数、クラス等を使用する時、異なるモジュールとの名前衝突を防ぐため、モジュール名はフルで記載する  

  ```
  import hoge  

  // 良い例  
  hoge.huga()  

  // 悪い例  
  huga()
  ```


- 型推論禁止  
Swiftでは型推論が可能ではあるが、コンパイル時間や可読性を考慮し型推論は原則禁止とし、型宣言を原則必須とする。

  ```
  // 良い例  
  var int_hoge: Int = 5  

  // 悪い例  
  var int_hoge = 5  
  ```

### 対応推奨  
対応必須では無いが可読性向上のため下記対応を願いたい

- **空白を入れる**  
    ```
    // カンマの後  
    [1, 2, 3]  

    // 演算子の前後  
    a + b  
    c / d  
    ```

- **一行をできるだけ短く**  
行数をできるだけ減らしたいところではあるが、一行が長くなりすぎると可読性が著しく低下するため、適度な文字数で改行を加え、折り返す  

- **不等号は左向きのみ使用**  
    ```
    0 < v && v <= 10  
    ```

- **インデントでのタブ文字禁止**  
タブキー使用時はスペース4文字に変換するよう設定  
Xcode>Preference>Text Editing から設定可能
